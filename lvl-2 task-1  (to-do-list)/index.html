I have made this to do web app during my internship at Tech-A-Intern.
This web app stores the task locally in the web browser.


How to Use this?

Add new Task:
  • Enter your task in the input field
  • Hit the Add Task button
Mark Task as Completed:
  • Click the Tick Mark present on the right side of the task.
Delete Task:
  • Click the Delete button present on the right side of the task.
Edit Task:
  • Click on the text of the task
  • Hit Enter button to save the modified task

Hit Escape button to close the Help Box.
